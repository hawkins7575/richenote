# PropertyDesk 사용자 가이드 (Beta)

> PropertyDesk v1.0.0-beta 베타 테스터를 위한 완전 가이드

## 🚀 빠른 시작

### 1. 회원가입 및 로그인

#### 회원가입
1. PropertyDesk 홈페이지 접속
2. **"회원가입"** 버튼 클릭
3. 필수 정보 입력:
   - **이름**: 본인 이름
   - **회사명**: 부동산 업체명 (선택사항)
   - **이메일**: 유효한 이메일 주소
   - **비밀번호**: 8자 이상 (12자+특수문자 권장)
4. **"회원가입"** 완료
5. 이메일 인증 후 로그인 가능

#### 로그인
1. 이메일과 비밀번호 입력
2. **"로그인"** 버튼 클릭
3. 자동으로 대시보드 페이지로 이동

### 2. 대시보드 둘러보기

로그인 후 첫 화면은 **대시보드**입니다:

- 📊 **매물 현황 차트**: 타입별, 가격대별 매물 분포
- 📈 **실시간 통계**: 총 매물 수, 평균 가격 등
- 🔥 **최근 활동**: 최근 등록된 매물 목록
- ⚡ **빠른 작업**: 매물 등록, 검색 등

## 🏠 매물 관리

### 매물 등록

#### 방법 1: 대시보드에서
1. 대시보드의 **"+ 매물 등록"** 버튼 클릭

#### 방법 2: 사이드바에서
1. 좌측 사이드바의 **"매물 관리"** → **"매물 등록"** 클릭

#### 등록 과정
1. **매물 등록 폼** 작성:
   ```
   📍 기본 정보
   - 제목: 매물 제목
   - 주소: 상세 주소
   - 매물 유형: 아파트/오피스텔/빌라/원룸/기타
   
   💰 가격 정보
   - 거래 유형: 매매/전세/월세
   - 가격: 해당 금액 (만원 단위)
   - 보증금: (월세인 경우)
   - 월세: (월세인 경우)
   
   📏 상세 정보
   - 층수: 해당 층 / 전체 층
   - 전용면적: ㎡ 단위
   - 방 개수: 개
   - 화장실 개수: 개
   
   📝 추가 정보
   - 설명: 매물 상세 설명
   ```

2. **"등록하기"** 버튼 클릭
3. 등록 완료 후 매물 목록에서 확인

### 매물 조회 및 검색

#### 전체 매물 보기
1. 사이드바 **"매물 관리"** 클릭
2. 등록된 모든 매물을 카드 형태로 확인

#### 실시간 검색
1. 상단 검색바에 검색어 입력:
   - 주소로 검색 (예: "강남구", "서초동")
   - 가격으로 검색 (예: "5억", "50000")
   - 매물 유형으로 검색 (예: "아파트", "오피스텔")

#### 필터 기능
1. **매물 유형 필터**: 아파트, 오피스텔, 빌라 등
2. **거래 유형 필터**: 매매, 전세, 월세
3. **가격대 필터**: 원하는 가격 범위 설정

### 매물 수정 및 삭제

#### 매물 수정
1. 매물 카드의 **"수정"** 버튼 클릭
2. 정보 수정 후 **"저장"** 클릭

#### 매물 삭제
1. 매물 카드의 **"삭제"** 버튼 클릭
2. 삭제 확인 후 완료

## 👥 팀 관리 (권한별 기능)

### 권한 체계
PropertyDesk는 4단계 권한 시스템을 사용합니다:

| 권한 | 설명 | 주요 기능 |
|------|------|-----------|
| **업체 대표** | 최고 관리자 | 모든 기능 + 팀원 관리 |
| **팀장/실장** | 팀 관리자 | 매물 관리 + 팀원 초대 |
| **중개사** | 실무자 | 매물 등록/수정 |
| **조회자** | 읽기 전용 | 매물 조회만 가능 |

### 팀원 초대 (팀장 이상)

#### 초대 과정
1. 사이드바 **"팀 관리"** 클릭
2. **"팀원 초대"** 버튼 클릭
3. 초대 정보 입력:
   ```
   📧 이메일: 초대할 사람의 이메일
   👤 역할: 중개사/조회자 중 선택
   📝 메모: 초대 메시지 (선택사항)
   ```
4. **"초대 발송"** 클릭
5. 이메일로 초대 링크 발송

#### 권한 관리
- **업체 대표**: 모든 역할에 초대 가능
- **팀장/실장**: 중개사, 조회자만 초대 가능
- **중개사/조회자**: 초대 기능 없음

### 팀원 관리

#### 팀원 목록 확인
1. **"팀 관리"** 페이지에서 현재 팀원 목록 확인
2. 각 팀원의 역할과 상태 표시

#### 역할 변경 (업체 대표만)
1. 팀원 목록에서 **"역할 변경"** 버튼 클릭
2. 새로운 역할 선택 후 저장

## 📊 분석 및 리포트

### 대시보드 차트

#### 매물 유형별 분포
- 아파트, 오피스텔, 빌라 등의 비율을 파이 차트로 표시
- 클릭하면 해당 유형 매물만 필터링

#### 가격대별 분포
- 1억 미만, 1-3억, 3-5억, 5억 이상으로 구분
- 바 차트로 시각화

#### 거래 유형별 현황
- 매매, 전세, 월세 비율
- 도넛 차트로 표시

#### 월별 등록 현황
- 최근 6개월간 매물 등록 추이
- 라인 차트로 트렌드 확인

### 실시간 통계
- **총 매물 수**: 현재 등록된 전체 매물
- **평균 가격**: 매물들의 평균 가격
- **이번 달 신규**: 이번 달 신규 등록 매물
- **활성 거래**: 진행 중인 거래 건수

## ⚙️ 설정 및 프로필

### 프로필 관리
1. 우상단 프로필 아이콘 클릭
2. **"프로필 설정"** 선택
3. 개인 정보 수정:
   - 이름 변경
   - 회사명 변경
   - 비밀번호 변경

### 업체 설정 (업체 대표만)
1. **"설정"** 메뉴 접속
2. 업체 정보 관리:
   - 업체명
   - 업체 주소
   - 연락처 정보

## 🔒 보안 및 개인정보

### 비밀번호 관리
- **강력한 비밀번호 사용**: 8자 이상, 특수문자 포함 권장
- **정기 변경**: 3개월마다 비밀번호 변경 권장
- **비밀번호 재설정**: 로그인 페이지에서 이메일로 재설정 가능

### 데이터 보안
- 모든 데이터는 암호화되어 저장
- 업체별 완전 분리된 데이터 관리
- 권한 기반 접근 제어로 보안 강화

## 📱 모바일 사용법

PropertyDesk는 모바일에서도 완벽하게 작동합니다:

### 모바일 브라우저 접속
1. 스마트폰 브라우저에서 PropertyDesk 접속
2. 자동으로 모바일 최적화 화면으로 전환

### 주요 모바일 기능
- **터치 최적화**: 버튼과 입력 필드가 터치에 최적화
- **반응형 차트**: 모바일에서도 차트 완벽 표시
- **빠른 검색**: 모바일에서도 실시간 검색 지원
- **간편 등록**: 터치로 간편한 매물 등록

## 🆘 문제 해결

### 자주 묻는 질문

#### Q: 로그인이 안 됩니다
**A**: 다음을 확인해주세요:
- 이메일 주소가 정확한지 확인
- 비밀번호 대소문자 확인
- 이메일 인증을 완료했는지 확인
- "비밀번호 재설정" 기능 사용

#### Q: 매물 등록이 안 됩니다
**A**: 권한을 확인해주세요:
- **조회자**: 매물 등록 불가
- **중개사 이상**: 매물 등록 가능
- 필수 입력 항목이 모두 채워졌는지 확인

#### Q: 팀원 초대가 안 됩니다
**A**: 다음을 확인해주세요:
- **팀장 이상** 권한이 있는지 확인
- 초대할 이메일 주소가 정확한지 확인
- 이미 가입된 이메일인지 확인

#### Q: 차트가 표시되지 않습니다
**A**: 다음을 시도해주세요:
- 브라우저 새로고침 (F5)
- 브라우저 캐시 삭제
- 다른 브라우저에서 접속
